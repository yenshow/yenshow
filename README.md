# YenShow Website

這是一個使用 Nuxt.js 框架開發的 YenShow 網站項目。

## 安裝與設定

請確保安裝所有依賴：

```bash
# npm
npm install
```

## 開發環境

在 `http://localhost:3000` 啟動開發伺服器：

```bash
# npm
npm run dev
```

## 生產環境

為生產環境建構應用程式：

```bash
# npm
npm run build
```

在本地預覽生產環境建構：

```bash
# npm
npm run preview
```

## 部署

請參閱 [Nuxt 部署文檔](https://nuxt.com/docs/getting-started/deployment) 以獲取更多信息。
